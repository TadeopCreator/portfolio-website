Docker commands:
    To update app cotainer:
    docker build -t my-django-app .

    To run django app container:
    docker run -d --network portfolio-network -p 8000:8000 my-django-app

Run el contenedor de postgres con: The postgres-container have to run to work app
docker run -d --name postgres-container --network portfolio-network -e POSTGRES_DB=portfolio -e POSTGRES_USER=tadeodeluca -e POSTGRES_PASSWORD=1234 postgres:latest

YOU HAVE TO MIGRATE DATABASE ON DOCKER TERMINAL (Debian environment) with: python manage.py migrate
